### 接口名称
更新分类排序

### 接口路径
POST /api/interface/up_cat_index

### 请求参数

#### 请求头

参数名称      | 类型   | 出现要求 | 描述
:-------------|:-------|:-------|:------------
Authorization | string | 必须     | Token xxxxxx

#### 请求体

参数名称    | 类型  | 出现要求 | 描述
:-----------|:------|:-------|:------
[]          | Array | 必须     | 列表
&emsp;id    | int   | 必须     | 分类id
&emsp;index | int   | 必须     | 修改后的索引

### 响应参数

#### 响应体

参数名称       | 类型 | 出现要求 | 描述
:--------------|:-----|:-------|:-----
modified_count | int  | 必须     | 更新条目数

### 响应码说明

响应码 | 说明
:------|:-------
40011  | 未登录
402    | 分类列表不完整
405    | 没有权限
