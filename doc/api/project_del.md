### 接口名称
删除项目

### 接口路径
POST /api/project/del

### 请求参数

#### 请求头

参数名称      | 类型   | 出现要求 | 描述
:-------------|:-------|:-------|:------------
Authorization | string | 必须     | Token xxxxxx
Content-Type  | string | 非必须   | application/json

#### 请求体

参数名称 | 类型 | 出现要求 | 描述
:--------|:-----|:-------|:----
id       | int  | 必须     | 项目id

### 响应参数

#### 响应体

参数名称     | 类型 | 出现要求 | 描述
:------------|:-----|:-------|:-----
n            | int  | 必须     | 匹配条目数
deletedCount | int  | 必须     | 删除条目数
ok           | int  | 必须     | 0, 1

### 响应码说明

响应码 | 说明
:------|:----
40011  | 未登录
400    | 没有权限
