### 接口名称
更新用户信息

### 接口路径
POST /api/user/update

### 请求参数

#### 请求头

参数名称      | 类型   | 出现要求 | 描述
:-------------|:-------|:-------|:------------
Authorization | string | 必须     | Token xxxxxx
Content-Type  | string | 非必须   | application/json

#### 请求体

参数名称     | 类型   | 出现要求 | 描述
:------------|:-------|:-------|:----
uid          | string | 必须     | 用户id
old_password | string | 必须     | 旧密码
password     | string | 必须     | 新密码

### 响应参数

#### 响应体

参数名称      | 类型 | 出现要求 | 描述
:-------------|:-----|:-------|:-----
modified_count | int  | 必须     | 更新条目数

### 响应码说明

响应码 | 说明
:------|:----------------
40011  | 未登录
400    | uid不能为空, uid不存在, 密码不能为空
402    | 旧密码错误
