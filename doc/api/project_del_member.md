### 接口名称
删除项目成员

### 接口路径
POST /api/project/del_member

### 请求参数

#### 请求头

参数名称 | 类型   | 出现要求 | 描述
:--------|:-------|:-------|:---------------
Cookie   | string | 必须     | _yapi_token=xxx

#### 请求体

参数名称   | 类型   | 出现要求 | 描述
:----------|:-------|:-------|:----
id         | string | 必须     | 分组名称
member_uid | int    | 必须     | 成员id

### 响应参数

#### 响应体

参数名称     | 类型 | 出现要求 | 描述
:------------|:-----|:-------|:-----
n            | int  | 必须     | 匹配条目数
deletedCount | int  | 必须     | 删除条目数
ok           | int  | 必须     | 0, 1

### 响应码说明

响应码 | 说明
:------|:-------
40011  | 未登录
400    | 项目成员不存在
405    | 没有权限
