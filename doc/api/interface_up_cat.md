### 接口名称
更新分类

### 接口路径
POST /api/interface/up_cat

### 请求参数

#### 请求头

参数名称      | 类型   | 出现要求 | 描述
:-------------|:-------|:-------|:------------
Authorization | string | 必须     | Token xxxxxx

#### 请求体

参数名称 | 类型   | 出现要求 | 描述
:--------|:-------|:-------|:----
id       | int    | 必须     | 分类id
name     | string | 非必须   | 分类名称
desc     | string | 非必须   | 分类描述

### 响应参数

#### 响应体

参数名称       | 类型 | 出现要求 | 描述
:--------------|:-----|:-------|:-----
modified_count | int  | 必须     | 更新条目数

### 响应码说明

响应码 | 说明
:------|:-----
40011  | 未登录
401    | 分类不存在
405    | 没有权限
